# pass-registration-system

### 项目简介

```
出入登记管理系统（H5前端系统）
使用 Vue + Mint-UI 开发
分两部分：
1、外来人员扫描二维码，并按要求填写相关信息，即可进出
2、管理员需要注册登陆，管理该管理员下的相关登记人员的信息、该管理员有专属自己的二维码、可以导出 excel 表功能等等
```
